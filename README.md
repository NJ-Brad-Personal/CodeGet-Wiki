# CodeGet-Wiki

Do you have code that you use in multiple projects?
Do you hate the thought of having multiple copies of code to maintain?
Does the thought that one of them is different scare you?
Are you afraid to make enhancements because you know you should update the rest.  (You know you will miss one or two)

CodeGet is NuGet for code.
It is tied to Git (and soon, other SCMs)
Through its UI you can add references to code from other repositories, directly into your solution.
Using it's build integration you can ensure that you are using the latest version of any code in your solution.
